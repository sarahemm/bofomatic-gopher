# bofomatic-gopher
This CGI allows (read-only) access to a BoF-O-Matic system from a gopher server
runing Gophernicus.

Installation:
1. Ensure you have the bofomatic gem installed.
1. Edit the bof-o-matic script to add your API key and API URL.
1. Copy the bof-o-matic script into your Gophernicus cgi-bin.
1. Copy the gophermap into the place in your gopher root where you want
   the BoF-O-Matic main menu to go.
1. Try it out!
